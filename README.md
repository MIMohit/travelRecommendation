# TravelBloom

TravelBloom is a simple travel recommendation website that suggests destinations based on user keywords such as **beach**, **temple**, or **country**.

## Features
- Keyword-based travel recommendations
- Dynamic data loading from a JSON file
- Image and description for each recommendation
- Home, About Us, and Contact Us pages
- Contact form with name, email, and message fields
- Responsive and modern UI

## Technologies
- HTML
- CSS
- JavaScript
- GitHub Pages

## How to Run
Open `travel_recommendation.html` in a browser or using Live Server.
